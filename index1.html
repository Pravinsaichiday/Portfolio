My Projects

Ultrasonic Security System

🏆 Awarded Best Project

Overview & Approach

This project is a cost-effective, Arduino-based home security system. To address the high cost of traditional systems, it uses an HC-SR04 ultrasonic sensor to detect motion. If an object breaches a set distance threshold, the Arduino triggers a buzzer or LED, providing an immediate and reliable alert for small spaces.

Key Features

  • Real-time motion detection via ultrasonic waves.
  • Instant sound/light alerts upon intrusion.
  • Adjustable sensitivity through simple code modification.
  • Compact, energy-efficient, and easily expandable design.
